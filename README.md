Course: CS419 - Digital Image and Video Analysis Institution: Sabancı University Semester: Fall 2024 Instructor: Erchan Aptoula

Assignment Overview Assignment 1: Registration of Retinal Images

**Objective: ** Register fluorescein angiography retinal images for accurate medical assessments.

Tasks:

Calculate the affine transformation matrix between image pairs. Perform the registration and visualize the aligned images. Compute the mean squared error (MSE) between the original and re-registered images and analyze the results.

Assignment 2: Convolution, Laplacian, and Connected Components

Objective:

Understand the mathematical properties of image processing operations and implement basic algorithms.

Tasks:

Prove the commutative, associative, and non-commutative properties of convolution and cross-correlation. Demonstrate that the Laplacian operator is rotation invariant. Analyze the linearity of a custom-designed filter and compute its convolution mask. Implement a two-pass algorithm for connected component extraction in binary images.

Assignment 3: Morphology, Noise Filtering, and Granulometry

Objective:

Apply advanced morphological operations, denoising techniques, and texture classification.

Tasks:

Implement morphological hole-filling on an image. Denoise an aerial image of Pompeii using a band-reject filter in the Fourier domain. Classify gravel types using granulometries and evaluate various distance metrics. Compare median filtering strategies (e.g., marginal, vector) for color images with salt-and-pepper noise and analyze their performance.

Assignment 4: Texture Classification with Variations

Objective:

Classify textures with variations in illumination and orientation using descriptor functions.

Tasks:

Design and test texture descriptors, including: Local Binary Patterns Gray Level Co-occurrence Matrices Fourier Transform-based descriptors Gabor filterbanks. Perform nearest-neighbor classification using 4 distance metrics: Euclidean, Manhattan, Chi-squared, and Mahalanobis. Analyze the effect of illumination variations on classification performance.
